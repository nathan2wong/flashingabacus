About

Flashing abacus is a web application designed to help students improve their mental arithmetic capabilities. Often, computer arithmetic exercises accompany the training many abacus students go through, and Flashing Abacus is a lightweight web-app that allows students to practice virtually anywhere.

To use this web interface, go to the application and click on one of the modules to begin (Currently supports addition and subtraction for all levels). From there, the user can specify the amount of numbers, the speed, and the number of digits to add or subtract.

More Information

If you would like to learn more about mental arithmetic / abacus and its applications in modern math, feel free to contact the creator or visit us at Jin's Mental Arithmetic Academy.
